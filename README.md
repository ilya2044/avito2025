# avito2025

## Makefile

| Команда       | Описание |
|---------------|----------|
| `make build`  | Компиляция сервиса в `bin/pr-reviewer` |
| `make up`     | Поднятие контейнеров с пересборкой образов |
| `make down`   | Остановка и удаление контейнеров |
| `make clean`  | Полностью очищает окружение (контейнеры, тома, бинарник) |
| `make migrate`| Применяет миграции к базе данных |
| `make run`    | Поднимает проект и автоматически выполняет миграцию, после чего выводит логи приложения |

---

## Запуск проекта

1. Собрать проект:

```bash
make build
```
2. Поднять контейнеры:
```bash
make up
```
3. Запустить сервис+миграции+просмотр логов:
```bash
make run
```
